Mini-projet3 PPE - Accueil

Ce site présente les résultats du mini-projet.

Ce tableau est généré automatiquement à l’aide d’un script Bash qui analyse plusieurs URL et extrait pour chacune d’elles le code de réponse HTTP, l’encodage du texte ainsi que le nombre de mots, afin de comparer l’accessibilité et la quantité de contenu des différentes pages web.
